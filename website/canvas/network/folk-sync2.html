Folk Sync 2 - Simplified Architecture

This is a demo of the new folk-sync2 attribute. It uses a simplified tree-based approach that directly mirrors the DOM structure in Automerge, with WeakMaps for O(1) node lookups.

Key improvements: Direct Automerge repo usage, tree structure instead of flat UUID map, and patches-based remote change handling.
